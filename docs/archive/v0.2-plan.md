## v0.2 功能（按此顺序构建）

### P0: Docker Compose 一键部署
- Dockerfile (Python + uvicorn)
- docker-compose.yml (ports + volumes + env vars)
- `docker compose up` 即可运行 Dashboard

### P1: Cost Manager（成本路由）
- 追踪每次 API 调用的 token 用量
- YAML 路由策略: free_first / quality_first / cheapest
- CLI: `relayos cost report` 查看用量统计

### P2: Dashboard 增强
- Workflow 实时时间线
- Agent 聊天面板（多轮对话）
