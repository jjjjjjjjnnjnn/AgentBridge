**RelayOS traite les conversations IA comme du code.** Fork, merge, tissez-les ensemble.

```bash
pip install relayos && relay
```

| Fonction | Description |
|----------|-------------|
| 🔀 Graphe de conversation | `/fork` `/merge` `/attach` |
| 🧠 Routage automatique | Modèles gratuits d'abord |
| 💰 Budget | Limites par tâche/jour/mois |
| ⌨️ TUI style OpenCode | Palette Ctrl+P, Tab, Chat |

### Commandes

```
/fork              
/merge id1 id2     
/attach id         
/remember k: val   
Ctrl+P             Palette de commandes
Ctrl+X N           Nouvelle session
Ctrl+X S           Liste des sessions
Ctrl+X G           Graphe de conversation
```

```bash
pip install relayos
```
