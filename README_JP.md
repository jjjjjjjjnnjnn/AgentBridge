---

**RelayOSはAI対話をコードのように扱います。**

```bash
pip install relayos && relay
```

| 機能 | 説明 |
|------|------|
| 🔀 会話グラフ | `/fork` `/merge` `/attach` |
| 🧠 自動ルーティング | 無料モデル優先 |
| 💰 予算保護 | タスク/日/月の制限 |
| ⌨️ OpenCodeスタイルTUI | Ctrl+Pパレット、Tab、チャット |

### コマンド

```
/fork              
/merge id1 id2     
/attach id         
/remember k: val   
Ctrl+X N           新規セッション
Ctrl+X S           セッション一覧
Ctrl+X G           会話グラフ
```

```bash
pip install relayos
```
