---

**RelayOS trata las conversaciones de IA como código.** Haz fork, merge y entrelázalas.

```bash
pip install relayos && relay
```

| Función | Descripción |
|---------|-------------|
| 🔀 Grafo de conversación | `/fork` `/merge` `/attach` |
| 🧠 Rutas automáticas | Modelos gratis primero |
| 💰 Presupuesto | Límites por tarea/día/mes |
| ⌨️ TUI estilo OpenCode | Paleta Ctrl+P, Tab, Chat |

### Comandos

```
/fork              
/merge id1 id2     
/attach id         
/remember k: val   
Ctrl+P             Paleta de comandos
Ctrl+X N           Nueva sesión
Ctrl+X S           Lista de sesiones
Ctrl+X G           Grafo de conversación
```

```bash
pip install relayos
```
