**RelayOS는 AI 대화를 코드처럼 다룹니다.**

```bash
pip install relayos && relay
```

| 기능 | 설명 |
|------|------|
| 🔀 대화 그래프 | `/fork` `/merge` `/attach` |
| 🧠 자동 라우팅 | 무료 모델 우선 |
| 💰 예산 보호 | 작업/일/월 한도 |
| ⌨️ OpenCode 스타일 TUI | Ctrl+P 팔레트, Tab, 채팅 |

### 명령어

```
/fork              
/merge id1 id2     
/attach id         
/remember k: val   
Ctrl+X N           새 세션
Ctrl+X S           세션 목록
Ctrl+X G           대화 그래프
```

```bash
pip install relayos
```
